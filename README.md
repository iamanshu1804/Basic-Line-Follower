# Basic-Line-Follower

# Purpose
Just a basic line follower built using webots and python. It is nothing fancy just me trying to learn about webots.

# Working:
It uses 3 IR sensors and basic python logic, if left sensor detects the black line then turn towards left and same for right and if no line is detected then read the last few readings of the 3 sensors and the one which last had the black line turn towards that side. The logic and code is completely written by me.
